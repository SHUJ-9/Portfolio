5-Step Moderation Process

A complete, repeatable framework for consistent content moderation decisions — from initial assessment through escalation and documentation.

Tier 1 — CriticalCSAM, terrorism, doxxing, direct threats → immediate removal + escalation
Tier 2 — High SeverityHate speech, graphic violence, harassment → remove + account action
Tier 3 — ModerateMisinformation, spam, copyright → restrict + notify user
Tier 4 — MonitorBorderline content, first-time minor breach → label + observe

Context Evaluation Matrix

Before finalizing any decision, content is evaluated across five dimensions: newsworthiness, educational value, satire, cultural context, and targeting of private individuals.
